# Copilot Syntax Highlighting

A VS Code theme that mimics the syntax highlighting style used by Microsoft Copilot when it displays code.

## Installation
1. Clone this folder.
2. Run `vsce package` to build a .vsix file.
3. Install the .vsix in VS Code.

## Development
Press F5 to launch a new VS Code window with the extension loaded.
Then select the theme:
Command Palette → Preferences: Color Theme → Copilot Syntax
